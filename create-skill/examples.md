# Create Skill — Examples

## Example 1: Code Review Skill

```
Create a skill for code review. It should follow our team standards: check for correctness, security, readability. Use a checklist format. Store as a project skill.
```

---

## Example 2: Commit Message Skill

```
Create a skill that generates conventional commit messages from git diffs. Include examples of feat, fix, chore, docs. Make it a personal skill.
```

---

## Example 3: Database Query Skill

```
Create a skill for querying our database schema. The agent should understand our table structure and generate safe, parameterized SQL. Include reference to our schema docs.
```

---

## Example 4: PDF Processing

```
Create a skill for extracting text and tables from PDFs. Use pdfplumber. Include workflow: analyze form, map fields, fill, validate. Add a validate.py script.
```
