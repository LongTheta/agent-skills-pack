# Example Prompts

## 1. Review an agent architecture

Use the ai-agent-architecture skill to review this design for a customer support agent:

- OpenAI model
- Pinecone for retrieval
- Internal order API tools
- LangGraph orchestration
- Kafka for events
- Kubernetes deployment
- No evaluation system yet

Assess all 7 layers, identify weakest points, and give a phased production-readiness plan.

## 2. Design from scratch

Use the ai-agent-architecture skill to design a minimal viable architecture for an internal DevOps assistant that:

- answers platform questions
- reads deployment status
- opens tickets
- suggests runbook actions

Keep prototype choices simple, then show how to harden it for production.

## 3. Gap analysis

Use the ai-agent-architecture skill to evaluate where this AI agent project is overengineered or underbuilt. Score each of the 7 layers from 1 to 10 and identify the weakest link.
