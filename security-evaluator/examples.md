# Security Evaluator — Examples

## Example 1: Tool Evaluation

Evaluate a tool for enterprise use.

```
Perform a security evaluation of Argo CD for use in our Kubernetes deployment pipeline. We're targeting FedRAMP Moderate. Include RBAC, audit logging, and secrets handling in the assessment.
```

---

## Example 2: Architecture Review

Assess an architecture for security risks.

```
Use the security-evaluator skill to review our proposed microservices architecture. We have 5 services, OIDC for auth, Vault for secrets. Data includes PII. Produce a security scorecard and identify gaps.
```

---

## Example 3: MCP Server Assessment

Evaluate an MCP server for security.

```
Evaluate the filesystem MCP server for security. We plan to use it in a development environment with access to project files. What are the risks and required mitigations?
```

---

## Example 4: CI/CD Workflow

Review a CI/CD workflow for security.

```
Use the security-evaluator skill to assess our GitHub Actions workflow for secrets exposure, supply chain risks, and compliance. We need audit evidence for SOC 2.
```
